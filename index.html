Git, Jenkins, Docker & Cloudflare Tunnel

This project showcases a complete DevOps pipeline For building, containerizing, and deploying a static website. The pipeline is triggered on every code change, automating the entire workflow from commit to a publicly accessible website.

Core Technologies

Git

Serves as the version control system. A `git push` to the repository initiates the CI/CD pipeline.

Jenkins

The automation server that fetches the latest code, runs the build, and manages the deployment process.

Docker

Containerizes the static website with a web server, ensuring a consistent and portable runtime environment.

Cloudflare Tunnel

Exposes the local Docker container to the internet securely, without needing a public IP or complex firewall rules.
